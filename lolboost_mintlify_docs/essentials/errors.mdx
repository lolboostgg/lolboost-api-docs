# Errors

| Code | Meaning | Fix |
| --- | --- | --- |
| 401 | Missing or invalid API key | Check the Authorization header |
| 422 | Validation error | Check required fields such as external_id, title and price |
| 429 | Too many requests | Wait and retry later |
| 500 | Server error | Contact LoLBoost support |
