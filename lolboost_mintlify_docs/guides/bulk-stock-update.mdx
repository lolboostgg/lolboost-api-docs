# Bulk stock update

Use bulk remove when multiple offers sold outside LoLBoost or when your stock changed.

```json
{
  "external_ids": [
    "offer-001",
    "offer-002",
    "offer-003"
  ]
}
```

Accounts use `POST /api/v1/accounts/bulk-remove`. Items use `POST /api/v1/items/bulk-remove`.
