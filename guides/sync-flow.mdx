Recommended seller sync flow:

1. Add in seller system, create on LoLBoost via API.
2. Save `lolboost_account_id` or `lolboost_item_id` from the response or created webhook.
3. Sold on LoLBoost, receive webhook and mark sold in seller system.
4. Sold outside LoLBoost, archive on LoLBoost by API.
