# LoLBoost Partner API

Use the Partner API to keep your seller inventory synchronized with LoLBoost.

<Cards>
  <Card title="Accounts" icon="gamepad" href="/api-reference/accounts/list-account-offers">Create, update, archive and bulk remove account offers.</Card>
  <Card title="Items" icon="gift" href="/api-reference/items/list-item-offers">Sync item offers such as gifts, services or stock based products.</Card>
  <Card title="Webhooks" icon="webhook" href="/webhooks/overview">Receive instant events when offers are created, archived or sold.</Card>
</Cards>

## Main sync flow

```text
Add in seller system, create on LoLBoost
Sold on LoLBoost, webhook to seller system
Sold outside LoLBoost, archive on LoLBoost by API
Bulk stock update, bulk archive on LoLBoost
```
