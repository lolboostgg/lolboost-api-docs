# Testing the API

1. Open any endpoint in the API Reference.
2. Click `Authorize`.
3. Enter your API key.
4. Click `Try it`.
5. Execute the request.

For a first test, create an account offer with this body.

```json
{
  "external_id": "mintlify-test-001",
  "title": "Mintlify Test Account",
  "game": "lol",
  "server": "EUW",
  "price": 999,
  "status": "listed"
}
```
