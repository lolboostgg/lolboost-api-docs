# Listing Rules

Listings must be accurate, deliverable, and updated. Sold or unavailable stock should be archived immediately. API sellers should use `external_id` to keep their own inventory synced with LoLBoost.

<Warning>Do not list the same account multiple times without automated stock sync. This can cause double sales.</Warning>
