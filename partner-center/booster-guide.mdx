# Booster Guide

Welcome to the LoLBoost booster team. This guide covers how to handle orders, communicate with customers, and stay compliant with platform rules. Read it fully before taking your first order, it's also where you'll come back to whenever a situation isn't clear.

## Order management

### Solo orders

| Rule | Details |
| --- | --- |
| Start within 30 minutes | Once you claim the order and login details are provided, begin promptly. |
| Message the customer first | Greet them in the Order Chat, confirm the order details, then start playing. |
| Stay active | Aim for 7-10 games per day. Fewer than that can count as inactivity. |
| Continue paused orders | If an order is paused, resume within 14 hours. Longer delays can be penalized. |

### DuoQ orders

| Rule | Details |
| --- | --- |
| Schedule through the Order Chat dashboard | Never schedule through the League client or Discord DMs. |
| Confirm playtimes | If you can't make an agreed time, open a Boosting Ticket early, missing playtimes without notice isn't allowed. |
| Use a correctly matched account | Your DuoQ account must be within 2 divisions of the customer's rank. |
| Never use the customer's account for DuoQ | Unless staff has explicitly approved it for that order. |

## Boosting rules

* Follow every order detail exactly: role, champions, summoner spells, streaming preference, offline mode, Solo Boost vs Hidden Duo, and any other notes on the order.
* Target at least a **70% winrate** across the boost.
* Never ask a customer to ignore order details or switch role/champions to make things easier on you.
* Keep in-game chat to essential shot-calling only.

<AccordionGroup>
  <Accordion title="Win Boost losses">
    Every loss during a Win Boost requires one additional win to compensate. The order isn't complete until the purchased number of net wins is reached.
  </Accordion>
  <Accordion title="Placements">
    5 placement matches require a 70% winrate. For fewer matches, any shortfall must be compensated with equivalent extra wins.
  </Accordion>
  <Accordion title="Promotions during a boost">
    Orders are fulfilled normally as the account ranks up. From Diamond 4 and above, one win is deducted after a promotion if more than one win is still pending, check with admins if you're unsure.
  </Accordion>
</AccordionGroup>

## Behavior

* **Zero tolerance for toxicity**: no flaming, griefing, intentional feeding, begging for tips, or rude behavior toward customers, teammates, or staff.
* **No private boosting or coaching.** Never add a customer on another platform or take work outside LoLBoost.
* If a customer asks you for private boosting or coaching, **report it immediately** via a Boosting Ticket, don't just decline quietly.
* Don't message admins privately about work, use Discord Tickets so there's a record.

***

## Account & performance

* **Finish and notify**: when a boost is done, message the customer and include the Trustpilot review link.
* **Send proof**: screenshots showing the in-game name, current rank, and win/loss record are required for Rank Boost, Win Boost, and Placement orders.
* **Use a working VPN** whenever the order requires it, and keep account activity looking natural.

### Dropping an order

Only drop an order if you have a valid Drop Token. Dropping without one, or asking the customer to drop it for you, is a violation (see fines below).

| Progress when dropped | Outcome |
| --- | --- |
| 1 win / 0 losses | Paid in full for progress |
| 1 win / 1 loss | €20 penalty |
| 1 win / 2 losses | €20 penalty |

Losses with a bad score require compensation games before the order can be considered complete. ACE or good-score losses don't.

## Fines

Fines are deducted directly from your payout. Admins may adjust a decision depending on proof provided and repeated behavior.

### Major violations (€50-€250)

| Violation | Fine |
| --- | --- |
| Private boosting, coaching, or accepting tips for extra services | €250 |
| Using bots, scripts, or other third-party programs | €200 |
| Customer account gets chat banned and admins aren't told immediately | €100 |
| Sharing Discord, phone number, or other contact info with a customer | €75 |
| Giving, selling, or passing an order to another booster or platform without approval | €50 |
| Causing a chat ban on the customer's account | €50 |

### Standard violations (€20-€25)

| Violation | Fine |
| --- | --- |
| Not using a working VPN when required | €25 |
| Bad behavior in-game or toward a customer | €25 |
| Buying, using, or changing items on a Solo order account without approval | €25 |
| Messaging people from the customer's account (Solo only) | €20 |
| Ignoring order details (champions, role, summoner spells, streaming, offline mode) | €20 |

### Minor violations (€10)

| Violation | Fine |
| --- | --- |
| Not using offline chat mode when required (Solo or DuoQ) | €10 |
| Dropping an order without a valid Drop Token | €10 |
| Asking the customer to drop the order for you | €10 |

### Minor operational violations (€5)

| Violation | Fine |
| --- | --- |
| Invalid delete-game request | €5 |
| Skipping the DuoQ account for API tracking on purpose | €5 |
| Accepting a DuoQ order without a suitable account ready | €5 |
| Accepting an order without messaging the customer | €5 |
| Unnecessary delay after accepting an order | €5 |
| Not sending proof when admins ask for it | €5 |
| Wrong or missing order status updates | €5 |
| Ignoring admin instructions on an active order | €5 |
